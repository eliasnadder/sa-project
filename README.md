# Senet Game

A Python terminal implementation of the ancient Egyptian board game Senet.

## Description

Senet is one of the oldest known board games, dating back to ancient Egypt around 3100 BCE. This project provides a terminal-based version of the game with AI support.

## Project Structure

- `main.py` - Entry point for the game
- `game.py` - Game state management
- `board.py` - Board representation and display
- `rules.py` - Game rules and logic
- `sticks.py` - Throwing sticks (dice equivalent)

## Requirements

- Python 3.x

## How to Run

```bash
python main.py
```
